## What changed

### `app/globals.css` (token migration)

Legacy var names preserved — every Tailwind utility (`bg-teal`, `text-amber`, `border-cream-2`, etc.) keeps working without code changes elsewhere; only the underlying hex value swapped.

### `app/layout.tsx` (font swap)

`Newsreader` import removed; `Fraunces` added with `style: ["normal", "italic"]` and `axes: ["SOFT", "opsz"]`. Loaded as a variable font (no `weight` array — variable fonts manage their own wght axis via CSS `font-weight`). For dramatic wordmark renderings, add `style={{ fontVariationSettings: "'opsz' 144, 'SOFT' 50" }}` at the wordmark site.

### `public/yentl-mark.svg` (new) + `public/yentl-y-mark.png` (deleted)

Vector replaces raster. Y-monogram with integrated checkmark, single fill `#2563EB`, two paths, viewBox 1400×1400. Source: `bold-cray-e9479f/.project/assets/logos/v5/yentl-mark.svg`. The PNG was 600×340 and embedded teal — deleting it avoids brand-inconsistent rendering.

### 4 file references updated

`app/page.tsx`, `components/session/source-picker.tsx`, `components/session/ingest-panes/mic-prerecord-pane.tsx`, `tests/source-picker.test.tsx`.
